# Coregix

Standalone extraction of the elastix-based pairwise raster alignment workflow from `vhrharmonize`.

Current scope:
- pairwise raster alignment CLI
- structural WorldView-to-LiDAR registration mode
- elastix/transformix-based transform estimation and application
- large-scene moving-grid output handling

The initial history in this repo was replayed from the elastix-focused commits in the source repository so the alignment work can evolve independently.
